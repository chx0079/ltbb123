# UI Enhancer Skill

This skill helps you improve the application's user interface by leveraging open-source design resources and modern UI trends found on GitHub.

## Capabilities

1.  **Search & Discover**: Identifies popular UI components and design systems on GitHub (e.g., shadcn/ui, Tailwind UI components).
2.  **Apply Modern Styles**: Updates CSS/Tailwind classes to follow modern design principles like Glassmorphism, Neumorphism, or Material Design 3.
3.  **Optimize UX**: Refactors component structures for better accessibility and responsiveness.

## Usage

When the user asks for UI improvements:
1.  Identify the component or page that needs enhancement.
2.  Search GitHub for relevant implementations or design inspirations.
3.  Apply the found patterns to the codebase, ensuring consistency with the existing design system.

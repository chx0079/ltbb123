# 中国象棋 - 技能版

一个基于HTML5 Canvas的中国象棋游戏，带有独特的技能系统。

## 特性

- 🎮 **人机对战**：使用Minimax算法实现AI，支持三种难度
- ⚡ **技能系统**：
  - **连击**（消耗2点能量）：本回合可移动两次
  - **换位**（消耗3点能量）：交换两个己方棋子的位置
- 📱 **响应式设计**：适配桌面和移动设备
- 🎨 **现代简约UI**：扁平化设计，清晰易读

## 游戏规则

### 基本规则
- 红方先手，双方轮流走棋
- 所有棋子遵循标准中国象棋移动规则
- 将死对方或将/帅被吃掉即获胜

### 技能系统
- 双方开局各有**3点能量**
- 能量有限，需要策略性使用
- 技能只能在己方回合使用
